[게시판 관리] 게시판 생성 시 유형은 무엇인가요?

ㆍ게시판 유형은 게시판 성격에 맞춰 "일반형 / 갤러리형 / 이벤트형 / 1:1게시판형"을 선택하여 생성할 수 있습니다.

ㆍ일반형 : 게시판 리스트에 제목만 노출되는 게시판입니다.

ㆍ갤러리형 : 게시판 리스트에 제목 이외에 이미지를 추가적으로 노출하는 게시판입니다.

ㆍ이벤트형 : 게시글 등록 시 기간(시작일, 종료일)을 설정할 수 있는 게시판입니다.

ㆍ1:1게시판형 : 문의와 답변을 1:1로 매칭하여 확인할 수 있는 게시판입니다.

   └ 게시글의 상태 값을 제공합니다. ("접수 / 답변대기 / 답변완료")


[게시판 관리] 신규 생성한 게시판은 쇼핑몰에 어떻게 연결하나요?
ㆍ신규 생성된 게시판은 게시판 리스트에 추가되며, [보기] 버튼을 클릭하여 페이지로 이동하거나 
   [PC] / [모바일] 버튼을 클릭하여 게시판의 URL을 클립보드에 복사할 수 있습니다.
ㆍ"디자인 > 디자인 관리 > 디자인 페이지 수정"에서 원하는 페이지 내에 이미지 또는 텍스트를 넣고
   게시판 URL을 링크를 처리하면, 신규 생성한 게시판으로 연결시킬 수 있습니다.
